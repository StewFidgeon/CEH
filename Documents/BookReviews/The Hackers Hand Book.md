## What specifically does the book cover?
Chapter 1 - The Anatomy of Financialdata Theft

Phase 1: Reconnaisance - target individuals by scraping social media (linked in etc) using Social Mapper, phishing casts a wide net over a target org, and malware in disguise targeting mobile devices. Target corporations using OSINT Open source intelligence using theHarvester, network mapping with nmap and Shodan, followed up with dark web credential hunting for employee creds found in breaches.

Phase 2: Initial access and exploitation - spear-phishing targetting specific individuals or departments, fake login pages (created using HTTrack website copier), malicious attachments via trojanized files (pdf, word, zip). Malware deployment such as banking trojans (TrickBotZeus/ZZbot), keyloggers (Agent Tesla), and Remote Access Trojans (DarkComet, NanoCore). Web Skimming aka a Magecart attack using JavaScript skimmers injected into checkout pages, and POS terminal hacking. Automated account takeover reusing passwords from previous data breaches using automated brute force login tools (Sentry MBA, SSNiper), CAPTCHA and security measures bypassed for credential stuffing attacks.

Phase 3: Data exfiltration and monetization - exfiltrating data uses dark web drops (Genesis Market, RussianMarket), TOR encrypted channels using SSH tunnels & VPN chains with C&C via Telegram, and cloud exfiltration. Stolen data monetized by selling fullz which are complete data packages of financial information, carding, money muling & laundering, conversion to crypto and washed through mixers and tumblers. Covering tracks and staying under the radar rely on anonymization via VPNs, TOR, and hosting services in countries with lax regulations.; timestomping to modify timestamps in logs and files, fileless malware living only in memory, or by deploying malware as a diversion to cover up their data theft.  The Carbanak cyber gang infiltrated via phishing, once in they deployed keyloggers to capture creds, moving laterally and gaining admin control, eventually learning how to dispense cash via ATM at set times to money mules, stealing over 1 Bn USD. Defending against this requires layering, MFA, endpoint detection on laptops and servers, AI based fraud detection, secure payment gateways that encrypt payment data during processing and transmission, and finally staying on top of threat inetelligence feeds.


Chapter 2 - The Cybercrime Arsenal

Marketplaces & forums - the dark web hosts most of the illicit marketplaces offering illegal activities and stolen data (genesis mart, RussianMarket, AlphaBay, Exploit.in, and Breach Forums). 

Tools and services offered - Malware & exploits, stolen credentials, zero-day vulnerabilities, botnets for hire, phishing kits, and money laundering services.

Open Source & underground hacking tools - theHarvester for gathering OSINT (email addresses,employee details, subdomains and publicly available info), Shodan to find exposed systems and IoT devices, Maltego for data mining and link analysis to help reveal relationships between bits of information.

Exploitation and privilege escalation - Metasploit is the industry standard tool for pen testing, having a vast library of exploits, payloads, and additional modules available, Cobalt Strike is commerical red teaming software for C&C, lateral movement, and post exploit activity, Mimikatz is used for credential harvesting from Windows systems,Evilginx2 for bypassing 2FA by acting as a reverse proxy on compromised devices, Social Engineering Toolkit for creating fake websites, phishing emails and credential harvesting attacks, and Modlishka for reverse proxy 2FA defeat similar to Evilginx2 but with more automation.
